# Usage

Guide d'utilisation de la plateforme web Aerium.

## Flux principal

1. Connexion.
2. Consultation dashboard.
3. Gestion des capteurs.
4. Suivi des alertes.
5. Export des rapports.

## Pages importantes

- Dashboard : synthese temps reel.
- Analytics : analyse historique.
- Sensors : CRUD capteurs.
- Alerts : pilotage des incidents.
- Reports : export PDF/CSV.
- Maintenance : planification technique.
- Admin : supervision globale.

## Bonnes pratiques

- Configurer des seuils par zone metier.
- Surveiller les alertes en continu.
- Exporter regulierement les rapports pour suivi.
